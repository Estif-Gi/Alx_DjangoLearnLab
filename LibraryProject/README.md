# LibraryProject

A simple Django blog application created as a learning project.

This repository contains a minimal blog app used for learning Django basics (models, views, templates, admin, and routing).

Quick start:

1. Create and activate a virtual environment
2. Install requirements: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Create a superuser: `python manage.py createsuperuser`
5. Start the dev server: `python manage.py runserver`

See `LibraryProject/README.md` for more details and examples.
